# README HW №10
### **Работа с ansible:**
На локальную ОС произведена установка ansible посредством pip

Развернуты интсансы appserver и dbserver с помощью terraform из окружения stage

Добавлены конфигурационные файлы ansible

Добавлены файлы инвентори и файл плейбук clone.yml

Осуществлено управление хостами с помощью ansible модулей ping, command, shell, systemd, service

Используя ansible модуль git произведено клонирование репозитория, затем клонирование репозитория с помощью плейбука clone.yml

После выполнения команды ```ansible app -m command -a 'rm -rf ~/reddit'``` произведено повторное клонирование плейбуком, результатом которого было сообщение об изменении состояния, т.к. предыдущей командой был удален каталог репозитория
